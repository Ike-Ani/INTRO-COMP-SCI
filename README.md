# Introduction to Java Programming

## Overview
This repository contains coursework, projects, and practice exercises completed for my Introduction to Java course. The class focused on foundational programming concepts, object-oriented programming principles, and problem-solving using Java.

The goal of this repository is to demonstrate my understanding of core programming fundamentals and my ability to write structured, maintainable Java code.

---

## Course Topics Covered
- Java syntax and program structure
- Variables and data types
- Conditional statements (if/else, switch)
- Loops (for, while, do-while)
- Methods and modular programming
- Object-Oriented Programming (OOP)
